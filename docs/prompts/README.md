## What are Prompts?

Prompts are detailed markdown documents that describe:
- Context and purpose of a feature
- Screen layouts and UI requirements
- Data models and relationships
- AI agent specifications (Phase 2+)
- Acceptance criteria

## Naming Convention

```
XX-[area]-[feature].md
```

Examples:
- `03-listings-apartments.md`
- `04-listings-cars.md`
- `07-saved-favorites.md`
